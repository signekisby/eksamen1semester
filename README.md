# Connected Starters – 1. semester eksamensprojekt

Dette repository indeholder koden til mit 1. semester eksamensprojekt på Multimediedesign.

Projektet er et responsivt website for Connected Starters, som har til formål at formidle deres tilbud til unge studerende og iværksættere i Region Sjælland på en overskuelig og indbydende måde.

## Teknologi
- HTML
- CSS (fælles.css)

## Indhold
Repository’et indeholder følgende sider:
- Forside (index.html)
- Om os (omos.html)
- Events (events.html)
- Kontakt (kontakt.html)

Der er arbejdet med semantisk HTML, responsivt layout og grundlæggende tilgængelighed.
